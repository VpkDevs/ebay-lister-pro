# SHIP CHARTER — First Ship (drafted 2026-07-20)

> Founder gets ONE edit pass. After confirmation, scope is locked for the 7-day First Ship run.

## Target buyer
Active eBay resellers doing 50–200 listings/month who lose hours to manual listing and underprice items for lack of sold-comp data.

## Core promise
Paste an item (or scrape it with the Chrome extension), and get it listed on eBay — priced from real sold comps — in under two minutes.

## Price point
**$29/month** (single paid tier for First Ship; Free tier stays as trial funnel, $69 tier hidden until after first dollar).

## Acquisition channel
Direct outreach to resellers the founder already knows / interacts with (r/Flipping and eBay-seller communities as Day-7 fallback only).

## The three features (v1 — nothing else)
1. eBay listing flow: import/scrape → AI metadata (Gemini) → sold-comp IQR pricing → publish to eBay.
2. Listing history + repricer (SQLite-backed) so users see and maintain their listings.
3. Chrome extension scraping into the dashboard.

## Hard stop list — will NOT exist in v1
- Shopify / WooCommerce / Etsy / Mercari / Poshmark cross-posting (dark-launched; UI de-emphasized)
- Accounting export, bin locations, analytics dashboards, feedback engine
- Brand polish, multi-tier pricing display, affiliate/annual plans
- **Fabricated testimonials and revenue claims on landing.html — MUST be removed before any public URL is shared. Non-negotiable.**

## First Ship gate
Live URL + one successful Stripe charge from a non-founder email + Sovereign Score ≥ 35.
