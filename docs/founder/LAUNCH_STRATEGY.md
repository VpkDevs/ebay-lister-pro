# Launch Strategy — eBay Multi-Channel Lister Pro

This document details the launch positioning, target channels selection, and launch milestones.

---

## 🎯 Positioning & Value Prop

Lister Pro is positioned as the **only 100% zero-dependency, self-hosted multi-channel listing platform for professional resellers**.

- **Primary message**: "Stop paying $50+/month for cross-listing apps. Run a lightning-fast, zero-dependency lister locally on your machine that watermarks images, parses live comp pricing using Gemini AI, and cross-posts to eBay, Shopify, WooCommerce, and Etsy in 90 seconds."
- **Key differentiators**:
  1. Zero external packages = zero bloat, instant startup, and absolute privacy.
  2. Local-first file database ensures your inventory history is yours.
  3. Integrated manifest V3 extension scrapes source items automatically.
