# Audience Map — eBay Multi-Channel Lister Pro

This document maps where target multichannel resellers, flippers, and e-commerce operators congregate online.

---

## 👥 Reseller Communities Directory

### 1. Reddit: r/flipping (1.2M subscribers)
- **Pain language**: Inventory management bottlenecks, multi-channel pricing errors, double selling risks, listing times, crop/watermark tools.
- **Mod Policy**: Strictly prohibits raw advertising. Story-driven, value-adds, and tool development diaries are permitted if they explain technical implementations.
- **Action**: Share a free self-hostable project write-up detailing the "why I built this zero-dependency multichannel lister."

### 2. Reddit: r/ebay (450K subscribers)
- **Pain language**: eBay seller hub changes, bulk listing policies, VeRO brand takedowns, price repricing.
- **Mod Policy**: Strictly monitored. Self-promotion must be declared.
- **Action**: Provide a helpful guide on "Avoiding VeRO list violations dynamically using AI checks," citing the local tool's brand blacklist.

### 3. Shopify Community Forums (500K+ users)
- **Pain language**: Syncing inventory levels from eBay, bulk cross-listing, product draft duplication.
- **Mod Policy**: Permissive of app announcements if solving a specific thread query.
- **Action**: Answer active threads seeking "free or cheap ways to sync eBay inventory to Shopify."

### 4. eBay Sellers Facebook Groups (100K+ combined members)
- **Pain language**: Cross-posting software fees (complaining about Vendoo/Crosslist monthly costs).
- **Mod Policy**: Group-dependent, usually permits self-promotion on designated threads.
- **Action**: Focus on the zero-cost self-hosted open-source model of the tool compared to expensive SaaS options.

## 🎯 Target Micro-Influencer List (1K - 50K followers)

1. **@ResellingVlogs** (Twitter/YouTube, 20K followers)
   - Beat: Reseller tools, sourcing guides, eBay tips.
   - Hook: Pitch how they can water-mark, spruce, and list items to eBay, Etsy, and Shopify in under 90 seconds.
2. **@PartTimeFlipper** (Twitter/Instagram, 15K followers)
   - Beat: Hustling side income via eBay/Mercari.
