eBay Integration Guide

  1. Create or log in to your eBay Developer Account.
  2. Under Application Keys, generate Production keys.
  3. Copy the App ID (Client ID) and Cert ID (Client Secret) and paste them below.
  4. Set your redirect URI (RuName OAuth Redirect URI) to: http://localhost:45900/api/auth/ebay/callback.

Method 1: Connect via OAuth (Recommended)

Save your Client ID, Secret, and RuName first, then click below to authorize directly on eBay.

Method 2: Manual Refresh Token

Shopify Integration Guide

  1. Log in to your Shopify Admin and go to Settings ➔ Apps and sales channels ➔ Develop apps.
  2. Click Create an app and name it "eBay Lister Pro".
  3. Under Admin API integration, select scopes: write_products, read_products, write_inventory, read_inventory.
  4. Install the app and copy your Admin API Access Token (starts with shpat_).

WooCommerce Integration Guide

  1. Go to WordPress Admin ➔ WooCommerce ➔ Settings ➔ Advanced ➔ REST API.
  2. Click Add key, select Description: "Lister Pro", Permissions: Read/Write.
  3. Generate and copy the Consumer Key (ck_...) and Consumer Secret (cs_...).

Etsy Integration Guide

  1. Navigate to Etsy Developers Console.
  2. Register a new app to obtain your **API Keystring (Client ID)**.
  3. Copy your **Shop ID** from your Etsy shop info.

Poshmark Extension Guide

Poshmark listings are automated via the Chrome Extension. No API credentials are required.

  1. Open Poshmark Login Page.
  2. Log in to your account.
  3. The extension companion will automatically link and display a green connection status. You can now use the extension to scrape or autofill listing forms!

Mercari Extension Guide

Mercari listings are automated via the Chrome Extension. No API credentials are required.

  1. Open Mercari Page.
  2. Log in to your account.
  3. The extension companion will automatically link and display a green connection status in the sidebar.
